# 📚 Library Management System

A simple console-based Library Management System built with Java.

This project was created to practice Object-Oriented Programming (OOP), file handling, collections, and application structure in Java.

---

## 🚀 Features

- Add books
- View all books
- Borrow books
- Return books
- Search books by title
- Automatic book ID generation
- Save books to `.txt` file
- Load books from file when the program starts

---

## 🛠️ Technologies Used

- Java
- ArrayList
- File Handling
- Object-Oriented Programming (OOP)
